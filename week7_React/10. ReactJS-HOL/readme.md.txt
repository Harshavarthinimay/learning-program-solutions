# OfficeSpaceRentalApp

This is a simple ReactJS project created as part of a hands-on lab to demonstrate how JSX, inline CSS, and dynamic rendering work in React.

The app displays a list of office space rental options and applies conditional styling to the rent amount.

---

## Features

- Uses JSX to create and render UI components
- Renders a list of office spaces with name, rent, and address
- Uses JavaScript expressions inside JSX
- Applies inline styling to color the rent:
  - Red if rent is below 60000
  - Green if rent is 60000 or above

---

## Technologies Used

- ReactJS
- JSX
- Inline CSS
- JavaScript (ES6)
- Node.js and NPM

---
