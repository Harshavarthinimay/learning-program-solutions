# EventExamplesApp

This ReactJS project demonstrates handling various events using React's event system, including synthetic events, inline functions, and custom event handling logic.

---

## Features

- Counter with Increment and Decrement buttons
  - "Increment" button triggers two actions:
    - Increases the counter
    - Displays a hello message in console
- "Say Welcome" button that takes an argument and shows an alert
- Button to handle a synthetic event with an alert: "I was clicked"
- Currency converter that converts Indian Rupees to Euro using a form submission event

---

## Technologies Used

- ReactJS (with functional components and hooks)
- JavaScript ES6
- Inline event handlers
- Synthetic events
- Controlled form inputs

---

## How to Run

### Prerequisites

- Node.js (v14 or higher)
- NPM (Node Package Manager)

### Steps

1. Clone or download the project folder
2. Open a terminal inside the project directory
3. Install dependencies and start the app:

```bash
npm install
npm start
